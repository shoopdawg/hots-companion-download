Installation Instructions

Because this is a free, open-source community tool, it is not signed with an enterprise certificate. Windows SmartScreen will flag it as an unrecognized app.

  1. Click Download and save the ZIP file.
  2. Right-click the ZIP and select Extract All...
  3. Open the extracted folder and double-click hots-companion.exe.
  4. When the blue "Windows protected your PC" screen appears, click More info.
  5. Click the Run anyway button at the bottom.

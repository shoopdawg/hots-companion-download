hots-companion

A lightweight stats overlay for Heroes of the Storm loading screens.

Download v1.9.1

Build: March 14, 2026

How to Use

  • Show/Hide: Click the floating toggle button.
  • Move UI: Hold the Alt key and drag the button or stats window to reposition them.
  • Interactive Tooltips: Hover over any player name to see their top hero win rates and detailed MMR.
  • Level Coloring: Player names change color based on their account level (Bronze, Silver, Gold, Platinum).
  • Game Mode: Ensure HotS is set to Windowed (Fullscreen).

Installation Instructions

Windows SmartScreen will flag this as an unrecognized app because it is an unsigned community tool.

  1. Click Download v1.9.1 and save the ZIP.
  2. Right-click the ZIP and select Extract All...
  3. Double-click hots-companion.exe.
  4. On the blue screen, click More info then Run anyway.
